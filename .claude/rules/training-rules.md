# Training Programming Rules

## Strength training

- Program around six movement patterns: squat, hip hinge, horizontal press, vertical press, horizontal pull, vertical pull
- RPE/RIR for load prescription. RPE 7-9 main lifts, RPE 6-8 accessories
- Default: each muscle group 2x/week
- Working sets per muscle group per week: start at 10-14 during a deficit. Only increase if recovery supports it
- Rest: 2-4 min compounds, 1-2 min accessories
- Overload priority: add reps within range, then add load, then add a set
- Deload: reduce volume 40-50% every 4-6 weeks, or reactively after two consecutive sessions with performance decline

## Running integration

- Running supports body composition, not race goals (unless stated otherwise)
- Most runs conversational pace (zone 2). Max 1 hard session per week
- Schedule runs away from heavy squat/deadlift days
- If running and lifting must share a day, lift first
- If legs are notably sore from gym work, swap run for a walk or skip it

## Deficit-specific

- Do not increase total volume while in a deficit
- Maintain intensity (load) as long as possible. Volume is the first thing to reduce
- Maintaining current strength IS progress during a cut
- If persistent fatigue, poor sleep, or mood changes last more than a week, flag it as potentially too aggressive a deficit

## Session structure

- Warm-up: 5-10 min general + specific warm-up sets
- 1-2 main compounds: 3-4 working sets each
- 2-4 accessories: 2-3 sets each
- Target: 45-75 minutes total
