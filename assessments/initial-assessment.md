## Old Routine Analysis

### What Was Good
- RPE-based intensity control (7 → 10 across 4-week blocks)
- Compound-first session structure
- Exercise rotation across mesocycles
- All six movement patterns covered across the week

### Issues and Gaps
- **Excessive session volume** — 30-40+ working sets per session (core + main + superset + two accessory blocks + WOD + bonus pump). During a deficit, this is unsustainable
- **CrossFit WODs as finishers** — high fatigue, low hypertrophy stimulus. Wrong tool for a cut
- **No deload** — every block ended at RPE 10 with no recovery week
- **Squat pattern undertrained** — never a primary lift
- **No accessory progression tracking**
- **Running was ad hoc** — scattered in WODs, not programmed

### What to Keep
- RPE-based load prescription
- Compound-first ordering
- Core work (reduced)

### What to Change
- Cut session volume to 15-20 working sets
- Remove metcon finishers — use structured running for conditioning
- Program real deloads
- Make squat a primary lift
- Separate running from lifting

## Risk Factors to Monitor

1. **Consistency is the #1 risk** — self-identified as the main challenge. Program must be adherence-friendly: fewer sessions that matter more, not many sessions creating guilt when missed
2. **Press-dominant profile** — ensure pulling volume ≥ pressing volume for shoulder health
3. **No deload history** — may have accumulated fatigue. First 2 weeks should be conservative (RPE 7) to assess current capacity
4. **Strength numbers are stale** — expect to work below old numbers initially. This is normal, not failure
5. **Deficit + restarting training** — don't add training stress and caloric stress simultaneously. Start deficit mild, tighten after 2-3 weeks when training is established

## Recommended Program Structure

### Training: 4 days/week, upper-lower split

**Why 4 and not 5-6:** Consistency is the weak link. 4 days is enough for 2x frequency per muscle group with manageable sessions. If a day gets missed, it's still 3 — which works. 5-6 days creates more opportunities to "fail" and feeds the guilt-skip cycle.

**Why upper-lower:** Simple, balanced, each session has a clear purpose. Easy to slot running around it. Familiar enough to follow without overthinking.

| Day | Session | Duration |
|-----|---------|----------|
| Mon | Upper A (press focus) | 50-60 min |
| Tue | Lower A (squat focus) | 50-60 min |
| Wed | Run (zone 2) or rest | 30-40 min |
| Thu | Upper B (pull focus) | 50-60 min |
| Fri | Lower B (hinge focus) | 50-60 min |
| Sat | Run (zone 2) | 30-40 min |
| Sun | Rest | — |

- Running on Wed/Sat avoids overlap with heavy squat (Tue) and hinge (Fri) days
- If a gym day is missed, shift the week — don't double up
- If two gym days are missed, do 3 sessions that week (full body each) and resume normally

### Running: 2x/week zone 2

- Keep at 5 km for the first 3 weeks (no increase while starting the new program + deficit)
- Increase by max 1 km/week after week 3 if recovery allows
- Cap at 8-10 km/week total during the cut
- If legs feel dead for gym sessions, reduce running first

### Nutrition: moderate deficit, protein-first

- Start at ~2500 kcal/day
- Protein ≥160 g/day (non-negotiable)
- Track protein only for the first 2 weeks — simplest approach that works
- Reassess calorie target after 2-3 weeks of weigh-ins
- Minimize alcohol (already doing this)

### Periodization

- **Weeks 1-2:** Baseline. Find working weights. RPE 7 on everything. Learn the program
- **Weeks 3-4:** Normal training. RPE 7-8 compounds, RPE 7 accessories
- **Week 5:** Deload — same exercises, 50% volume, RPE 6
- **Weeks 6-8:** Push. RPE 8-9 compounds, RPE 7-8 accessories
- **Week 9:** Deload
- Reassess program at week 9 based on progress data
